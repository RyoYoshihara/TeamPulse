# 7. 補足

このタスク一覧は、MVP開発を素早く進めるために、  
「まず動くものを作り、その後に洗練する」前提で整理している。

特に以下が重要となる。

- 先にER設計を固定する
- 先に共通レスポンスと認証を作る
- 一覧画面を共通化する
- アサイン率と工数の集計ロジックを早めに決める
